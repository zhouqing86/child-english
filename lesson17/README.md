# Lesson 17: Transportation

## 主题

交通工具。

## 本课目标

- 让孩子认识常见交通工具和相关动作
- 建立 car、bus、bike、train、buckle up 等高频表达
- 把英语带入出行和路上观察情境

## 建议学习周期

2 周。

## 本课核心句型

- This is a car.
- This is a bus.
- This is a bike.
- This is a train.
- Go by car.
- Ride the bike.
- Buckle up.
- The light is green.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 找车游戏、角色扮演和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 出门坐车前播放 3 到 5 条句子音频。
2. 在路上观察真实车辆并重复这些句子。
3. 让孩子通过指认、模仿或回答来参与。
4. 回家后快速复习 2 分钟。
