# Lesson 17 Sentences

## 必须掌握

1. This is a car.
2. This is a bus.
3. This is a bike.
4. This is a train.
5. This is a taxi.
6. Go by car.
7. Ride the bike.
8. Wait for the bus.
9. The train is coming.
10. Get in the car.
11. Buckle up.
12. The light is red.
13. The light is green.
14. Stop the car.
15. Let's go home.
16. Fast train.

## 可扩展

1. This is a plane.
2. This is a boat.
3. The bus is big.
4. The bike is small.
5. The car is fast.
6. We are on the bus.

## 替换练习

- This is a car. / This is a bus. / This is a train.
- The light is red. / The light is green.
- The bus is big. / The bike is small.
- Go by car. / Ride the bike.
