# Lesson 17 Practice

## Game 1: Vehicle Hunt

出门时找交通工具：

- This is a car.
- This is a bus.
- This is a bike.

## Game 2: Toy Road

用玩具车和轨道练习：

- Go by car.
- Ride the bike.
- The train is coming.

## Game 3: Red Light, Green Light

做颜色反应游戏：

- The light is red.
- The light is green.
- Stop the car.

## Game 4: Car Seat Routine

上车时固定练习：

- Get in the car.
- Buckle up.
- Let's go home.

## Game 5: Replace One Vehicle

替换练习：

- This is a train.
- This is a taxi.
- This is a plane.

## 两周建议

### Day 1 到 Day 3

只用下面 5 句：

- This is a car.
- This is a bus.
- This is a bike.
- Get in the car.
- Buckle up.

### Day 4 到 Day 7

增加 5 句：

- This is a train.
- This is a taxi.
- Wait for the bus.
- The train is coming.
- Fast train.

### Day 8 到 Day 14

加入问答和替换：

- Go by car.
- Ride the bike.
- The light is red.
- The light is green.
- Let's go home.
