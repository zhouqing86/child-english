# Lesson 13 Parent Guide

## 本课最重要的原则

水果和零食课最好贴着真实食物学，孩子能看见、摸到、吃到，理解会更快。

## 推荐做法

1. 洗水果时自然说 Wash the grapes. / Peel the banana.
2. 让孩子在几种水果中做选择。
3. 用 It's sweet. / It's sour. 帮孩子表达口感。
4. 用 One for you. / One for me. 练习分享。

## 不推荐做法

- 只看图片不接触真实水果
- 一次塞太多水果名称
- 逼孩子每次都完整说长句
- 把食物场景变成考试场景

## 如何判断有效

下面这些都算有效学习：

- 孩子能在几种水果中做选择
- 孩子能说出 banana、apple、grapes 等关键词
- 孩子能回应 sweet / sour
- 孩子在加餐时主动冒出一句请求句

## 本课的真实目标

孩子在自然吃东西和分享食物时开始稳定使用一批高频英语句子。
