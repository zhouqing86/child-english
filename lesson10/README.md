# Lesson 10: Action Commands

## 主题

动作指令。

## 本课目标

- 让孩子快速听懂并执行基础动作指令
- 建立 stand、sit、jump、run、stop 等高频动词
- 通过 TPR 游戏提升听力和反应速度

## 建议学习周期

2 周。

## 本课核心句型

- Stand up.
- Sit down.
- Jump.
- Run.
- Walk slowly.
- Stop.
- Turn around.
- Good moving.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 动作游戏、反应游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 做动作游戏前播放 3 到 5 条句子音频。
2. 先让孩子执行动作，再鼓励跟说。
3. 每次活动控制在 5 到 10 分钟。
4. 睡前快速复习 2 分钟。
