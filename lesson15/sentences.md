# Lesson 15 Sentences

## 必须掌握

1. This is a dog.
2. The dog says woof.
3. This is a cat.
4. The cat says meow.
5. This is a bird.
6. The bird says tweet.
7. This is a cow.
8. The cow says moo.
9. This is a duck.
10. The duck says quack.
11. The rabbit hops.
12. The fish swims.
13. The horse runs.
14. I like the lion.
15. Which animal is it?
16. Good animal sound.

## 可扩展

1. This is a sheep.
2. The sheep says baa.
3. This is a frog.
4. The frog jumps.
5. The monkey climbs.
6. I like the panda.

## 替换练习

- This is a dog. / This is a cat. / This is a duck.
- The dog says woof. / The cat says meow.
- The rabbit hops. / The fish swims. / The horse runs.
- I like the lion. / I like the panda.
