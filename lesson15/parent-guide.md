# Lesson 15 Parent Guide

## 本课最重要的原则

动物课要优先借助孩子感兴趣的东西，例如玩具、绘本、手偶和声音模仿。

## 推荐做法

1. 先让孩子模仿动物叫声，再带出整句。
2. 一次只学少量动物，反复出现。
3. 把动物动作也带上，例如 hops、swims、runs。
4. 利用绘本和玩偶做问答游戏。

## 不推荐做法

- 一次性堆很多动物名称
- 只拿单词卡片，不做声音和动作
- 孩子模仿声音时立刻纠正太多
- 用太复杂的动物知识解释打断节奏

## 如何判断有效

下面这些都算有效学习：

- 孩子能把动物和声音连起来
- 孩子能通过动作表示不同动物
- 孩子能说出一两个完整句
- 孩子愿意主动问或答 which animal

## 本课的真实目标

孩子能在有趣的动物情境里自然开口，增强对英语的兴趣和熟悉感。
