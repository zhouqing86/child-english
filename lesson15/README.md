# Lesson 15: Animals and Sounds

## 主题

动物与声音。

## 本课目标

- 让孩子在玩具、绘本和视频中认识常见动物
- 建立动物名称和拟声表达的连接
- 利用孩子天然感兴趣的内容提高开口意愿

## 建议学习周期

2 周。

## 本课核心句型

- This is a dog.
- The dog says woof.
- This is a cat.
- The cat says meow.
- This is a duck.
- The duck says quack.
- Which animal is it?
- Good animal sound.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 模仿游戏、找动物游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 看绘本或玩动物玩具前播放 3 到 5 条句子音频。
2. 边看边说动物名称和声音。
3. 让孩子先模仿声音，再逐步跟说完整句。
4. 睡前快速复习 2 分钟。
