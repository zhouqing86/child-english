# Lesson 6: Toys and Clean Up

## 主题

玩具与房间收纳。

## 本课目标

- 让孩子在玩耍中学习最自然的玩具表达
- 建立 play、share、pick up、clean up 等高频用语
- 把英语放进整理房间的真实流程中

## 建议学习周期

2 周。

## 本课核心句型

- Let's play.
- I want the ball.
- Throw the ball.
- Catch the ball.
- Build a tower.
- Pick up your toys.
- Put it in the box.
- Clean up, please.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 玩耍游戏、收纳游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 玩玩具前播放 3 到 5 条句子音频。
2. 在投球、搭积木、收玩具时反复说这些句子。
3. 让孩子边玩边跟说，不强求完整复述。
4. 睡前快速复习 2 分钟。
