# Lesson 20 Practice

## Game 1: Home Tour

在家里边走边说：

- This is a table.
- This is a chair.
- This is a bed.

## Game 2: Open and Close

用门窗做动作：

- Open the door.
- Close the door.
- Open the window.
- Close the window.

## Game 3: Light On, Light Off

练习开关：

- Turn on the light.
- Turn off the light.

## Game 4: Find the Bag

找家里物品：

- Where is the bag?
- It's in the box.

## Game 5: Replace One Item

替换练习：

- This is a sofa.
- This is a lamp.
- This is a pillow.

## 两周建议

### Day 1 到 Day 3

只用下面 5 句：

- This is a table.
- This is a chair.
- This is a bed.
- Open the door.
- Turn on the light.

### Day 4 到 Day 7

增加 5 句：

- This is a door.
- This is a window.
- Close the door.
- Sit on the chair.
- Go to bed.

### Day 8 到 Day 14

加入问答和替换：

- Open the window.
- Close the window.
- Turn off the light.
- Where is the bag?
- It's in the box.
