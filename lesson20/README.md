# Lesson 20: Things at Home

## 主题

家里的常见物品。

## 本课目标

- 让孩子能认识并使用家里最常见的物品表达
- 建立 table、chair、door、window、light 等核心词
- 通过真实家居动作强化理解和记忆

## 建议学习周期

2 周。

## 本课核心句型

- This is a table.
- This is a chair.
- This is a bed.
- This is a door.
- This is a window.
- Open the door.
- Turn on the light.
- Where is the bag?

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 找物游戏、动作游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 在家里找物品前播放 3 到 5 条句子音频。
2. 通过真实开门、开灯、找包等动作重复这些句子。
3. 让孩子边找边说，不追求一次全会。
4. 睡前快速复习 2 分钟。
