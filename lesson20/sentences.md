# Lesson 20 Sentences

## 必须掌握

1. This is a table.
2. This is a chair.
3. This is a bed.
4. This is a door.
5. This is a window.
6. Open the door.
7. Close the door.
8. Open the window.
9. Close the window.
10. Put it on the table.
11. Sit on the chair.
12. Go to bed.
13. Turn on the light.
14. Turn off the light.
15. Where is the bag?
16. It's in the box.

## 可扩展

1. This is a sofa.
2. This is a lamp.
3. This is a shelf.
4. This is a pillow.
5. This is a blanket.
6. The room is bright.

## 替换练习

- This is a table. / This is a chair. / This is a bed.
- Open the door. / Close the door.
- Turn on the light. / Turn off the light.
- It's in the box. / Put it on the table.
