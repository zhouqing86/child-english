# Lesson 18: Shopping and Choice

## 主题

购物与选择。

## 本课目标

- 让孩子在购物场景中进行简单选择和表达偏好
- 建立 cart、money、buy、this one、that one 等高频表达
- 提高孩子在外部真实情境中的英语参与度

## 建议学习周期

2 周。

## 本课核心句型

- Let's go shopping.
- We need milk.
- Put it in the cart.
- Do you like this one?
- I like this one.
- Which one do you want?
- Here is the money.
- Let's buy it.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 选择游戏、角色扮演和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 去超市或玩购物游戏前播放 3 到 5 条句子音频。
2. 在真实选择和拿商品时重复这些句子。
3. 让孩子通过指、拿、选来参与。
4. 回家后快速复习 2 分钟。
