# Lesson 18 Practice

## Game 1: Shopping Basket

用家里的物品做购物游戏：

- We need milk.
- We need eggs.
- Put it in the cart.

## Game 2: This One or That One

拿两样商品练习：

- Do you like this one?
- Which one do you want?
- I want this one.

## Game 3: Cashier Role Play

角色扮演：

- How much is it?
- Here is the money.
- Thank you.

## Game 4: Buy or Put Back

选择商品练习：

- Let's buy it.
- Put it back.
- It's too expensive.

## Game 5: Replace One Item

替换练习：

- We need milk.
- We need bread.
- We need apples.

## 两周建议

### Day 1 到 Day 3

只用下面 5 句：

- Let's go shopping.
- We need milk.
- Put it in the cart.
- I like this one.
- Thank you.

### Day 4 到 Day 7

增加 5 句：

- We need eggs.
- Do you like this one?
- Which one do you want?
- I want this one.
- Let's buy it.

### Day 8 到 Day 14

加入问答和替换：

- That one is big.
- This one is small.
- How much is it?
- Here is the money.
- Good choosing.
