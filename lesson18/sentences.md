# Lesson 18 Sentences

## 必须掌握

1. Let's go shopping.
2. We need milk.
3. We need eggs.
4. Put it in the cart.
5. Do you like this one?
6. I like this one.
7. That one is big.
8. This one is small.
9. Which one do you want?
10. I want this one.
11. How much is it?
12. Here is the money.
13. Thank you.
14. Bye.
15. Let's buy it.
16. Good choosing.

## 可扩展

1. We need bread.
2. We need apples.
3. Put it back.
4. It's too expensive.
5. I want that one.
6. The cart is full.

## 替换练习

- We need milk. / We need eggs. / We need bread.
- I like this one. / I want that one.
- This one is small. / That one is big.
- Put it in the cart. / Put it back.
