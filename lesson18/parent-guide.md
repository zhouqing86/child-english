# Lesson 18 Parent Guide

## 本课最重要的原则

购物课最重要的是“参与”，不是把超市变成考试现场。

## 推荐做法

1. 让孩子承担少量真实任务，例如拿 milk、放进 cart。
2. 给孩子两个选择，练习 this one / that one。
3. 用简单句反复说，不要现场讲太多。
4. 如果不方便去超市，也可以在家做角色扮演。

## 不推荐做法

- 在超市里一口气灌很多新词
- 孩子犹豫时连续追问
- 用购物时间强迫完整复述
- 只用中文指挥，学完后再单独“复盘英语”

## 如何判断有效

下面这些都算有效学习：

- 孩子能在两样商品中做选择
- 孩子能听懂 put it in the cart 等句子
- 孩子能说出 milk、eggs、this one 等关键词
- 孩子能在角色扮演中说 thank you 或 here is the money

## 本课的真实目标

孩子在真实选择和购买情境里开始用英语表达偏好与需求。
