# Lesson 8: Numbers and Counting

## 主题

数字与数量。

## 本课目标

- 让孩子在真实点数场景中掌握基础数量表达
- 建立 one、two、more、how many 等核心句型
- 通过拿物、数物增强英语理解和输出

## 建议学习周期

2 周。

## 本课核心句型

- Count with me.
- One apple.
- Two bananas.
- How many?
- I see one.
- Give me two.
- One more.
- Good counting.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 点数游戏、给物游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 拿积木、水果或玩具先播放 3 到 5 条句子音频。
2. 在真实点数和分物时同步说英语。
3. 让孩子先拿、先摆，再逐步开口。
4. 睡前快速复习 2 分钟。
