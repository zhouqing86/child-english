# Lesson 11 Parent Guide

## 本课最重要的原则

情绪课不是为了考试孩子会不会说 happy，而是让孩子在真实感受里有英文表达出口。

## 推荐做法

1. 在真实情绪发生时自然命名，不要事后抽问。
2. 多用 Are you ...? / I'm ... 这种稳定句型。
3. 把安抚句也一起带上，例如 It's okay. / I need a hug.
4. 允许孩子先做表情、点头，再慢慢开口。

## 不推荐做法

- 孩子哭时强行让他重复句子
- 拿情绪词做机械配对题太久
- 把 angry、sad 当成必须立刻纠正的“坏情绪”
- 频繁翻译和追问原因，打断表达

## 如何判断有效

下面这些都算有效学习：

- 孩子能通过表情或点头理解情绪词
- 孩子能说出 happy、sad 等关键词
- 孩子在需要安抚时主动说 hug
- 孩子能在真实场景里用一句表达状态

## 本课的真实目标

孩子开始把英语用于表达感受和需求，而不是只用于认物和执行指令。
