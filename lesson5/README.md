# Lesson 5: Family Time

## 主题

家人称呼与家庭互动。

## 本课目标

- 让孩子在家里自然使用对家人的称呼
- 建立 love、hug、kiss、hello、bye-bye 等互动表达
- 把英语放进最亲密、最稳定的家庭情境中

## 建议学习周期

2 周。

## 本课核心句型

- This is Mommy.
- This is Daddy.
- I love Mommy.
- I love Daddy.
- Come to Mommy.
- Come to Daddy.
- Give Daddy a hug.
- Say hello.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 动作游戏、互动游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 家人见面和告别时播放 3 到 5 条句子音频。
2. 在拥抱、打招呼、视频通话时自然使用这些句子。
3. 鼓励孩子先用动作回应，再逐步开口。
4. 睡前快速复习 2 分钟。
