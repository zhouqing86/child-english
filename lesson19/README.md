# Lesson 19: Time and Daily Order

## 主题

时间与日常顺序。

## 本课目标

- 让孩子理解 morning、night、now、later 等日常时间词
- 建立 first、then、next 这样的顺序表达
- 帮助孩子用英语理解日常流程和过渡

## 建议学习周期

2 周。

## 本课核心句型

- It's morning.
- It's afternoon.
- It's night.
- It's time to eat.
- First wash your hands.
- Then let's eat.
- Later we will read.
- It's bedtime.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 流程排序游戏、角色扮演和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 在起床、吃饭、洗澡、睡前等节点播放 3 到 5 条句子音频。
2. 只结合当天真实流程使用这些句子。
3. 让孩子通过动作和排序来理解顺序。
4. 睡前快速复习 2 分钟。
