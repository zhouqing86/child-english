# Lesson 19 Parent Guide

## 本课最重要的原则

时间顺序课要贴着真实日程说，孩子才会真正理解 now、later、first、then 的含义。

## 推荐做法

1. 在真实流程切换时说 It's time to ...。
2. 用固定顺序句带孩子理解先后。
3. 反复在同样的生活节点使用同样的句子。
4. 可以用图片或卡片做简单排序辅助理解。

## 不推荐做法

- 抽象讲时间概念太久
- 一次引入太多时间词
- 脱离真实生活只做纸面练习
- 把 before、after 等讲成语法课

## 如何判断有效

下面这些都算有效学习：

- 孩子能听懂流程转换指令
- 孩子能按顺序做动作
- 孩子能说出 morning、night、bedtime 等关键词
- 孩子能在日常流程中主动补半句

## 本课的真实目标

孩子逐步用英语理解自己一天的节奏和顺序，而不是只记零散词汇。
