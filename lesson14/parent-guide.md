# Lesson 14 Parent Guide

## 本课最重要的原则

餐桌课的价值在于高频重复。三餐每天都有，所以句子要固定、稳定、持续使用。

## 推荐做法

1. 开饭前固定说 Sit at the table. / It's breakfast time.
2. 递餐具时固定说 Use your spoon. / This is your bowl.
3. 餐中让孩子练习简单请求，例如 I want soup.
4. 餐后固定说 I'm full. / All done. / Wipe the table.

## 不推荐做法

- 一边吃饭一边讲很多新单词
- 孩子刚要吃就要求长句复述
- 只在“上课”时说，平时不用
- 把吃饭变成语言考核

## 如何判断有效

下面这些都算有效学习：

- 孩子能按句子拿或用餐具
- 孩子能说 spoon、bowl、soup 等关键词
- 孩子能用 I want ... 表达需求
- 孩子能在饭后主动说 all done 或 I'm full

## 本课的真实目标

孩子在每天最稳定的三餐流程里，逐步建立一套可直接使用的英语表达。
