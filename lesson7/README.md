# Lesson 7: Colors and Shapes

## 主题

颜色与形状。

## 本课目标

- 让孩子在真实物品上识别颜色和基础形状
- 建立 red、blue、circle、square 等高频表达
- 开始练习简单观察和选择句型

## 建议学习周期

2 周。

## 本课核心句型

- What color is it?
- It is red.
- It is blue.
- Show me the red one.
- This is a circle.
- This is a square.
- This is a triangle.
- Good matching.

## 文件说明

- sentences.md: 核心句子和替换练习
- vocabulary.md: 由句子提取出的高频词汇
- dialogues.md: 家长和孩子可直接演练的微对话
- practice.md: 找物游戏、配对游戏和复习活动
- parent-guide.md: 家长使用说明
- audio-manifest.json: 音频生成清单

## 建议每日流程

1. 用积木、贴纸、玩具先播放 3 到 5 条句子音频。
2. 在真实物品上反复指认颜色和形状。
3. 让孩子先指、先拿，再逐步开口。
4. 睡前快速复习 2 分钟。
