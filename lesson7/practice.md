# Lesson 7 Practice

## Game 1: Color Hunt

在家里找颜色：

- Find a red ball.
- Show me the blue one.
- I like the yellow star.

## Game 2: Shape Box

拿形状积木或卡片练习：

- This is a circle.
- This is a square.
- This is a triangle.

## Game 3: Same or Different

准备两样物品练习：

- Match the same color.
- Same shape.
- Different color.

## Game 4: Big or Small

拿两个大小不同的物品练习：

- Big circle.
- Small square.

## Game 5: Replace One Color

替换练习：

- It is red.
- It is blue.
- It is green.

## 两周建议

### Day 1 到 Day 3

只用下面 5 句：

- What color is it?
- It is red.
- It is blue.
- Show me the red one.
- Good matching.

### Day 4 到 Day 7

增加 5 句：

- It is yellow.
- It is green.
- This is a circle.
- This is a square.
- Find a red ball.

### Day 8 到 Day 14

加入问答和替换：

- This is a triangle.
- Big circle.
- Small square.
- Match the same color.
- Different color.
