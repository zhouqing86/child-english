# Lesson 7 Parent Guide

## 本课最重要的原则

颜色和形状要尽量贴着真实物品学，不要只靠抽象卡片。

## 推荐做法

1. 在玩具、衣服、食物上自然问 What color is it?
2. 用积木或图卡带孩子说 This is a circle. / This is a square.
3. 把找颜色、找形状做成小游戏，而不是背诵。
4. 同一天只突出 1 到 2 个颜色或形状重点。

## 不推荐做法

- 一次把所有颜色和形状都灌进去
- 脱离真实物品死记单词
- 不停要求孩子翻译中文
- 因为孩子说错就频繁打断

## 如何判断有效

下面这些都算有效学习：

- 孩子能正确指出颜色
- 孩子能从多个东西里找出目标物品
- 孩子能说出 red、blue、circle 等关键词
- 孩子能用 this is a ... 补半句

## 本课的真实目标

孩子开始具备用英语观察和分类身边物品的能力。
