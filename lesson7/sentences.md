# Lesson 7 Sentences

## 必须掌握

1. What color is it?
2. It is red.
3. It is blue.
4. It is yellow.
5. It is green.
6. Show me the red one.
7. Show me the blue one.
8. This is a circle.
9. This is a square.
10. This is a triangle.
11. Big circle.
12. Small square.
13. I like the yellow star.
14. Find a red ball.
15. Match the same color.
16. Good matching.

1. It is orange.
2. It is purple.
3. This is a rectangle.
4. This is a heart.
5. Same shape.
6. Different color.
